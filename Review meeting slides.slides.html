variables = ['x']
odes      = [Poly(-1, 'x', 1, variables)] # [-x]

C0 = Reach(
    variables,  # List of variables
    odes,       # ODEs 
    [(4,5)],    # Initial conditions
    5,          # Time horizon
    0.3,        # Step size
    order=10,   # Taylor model order
)

Three-valued signal monitoring¶

We want to extract a three-valued $$\varphi : \mathbb R_{\geq 0} \to \{ \texttt{True}, \texttt{False}, \bot \}$$ from a Taylor model.

For atomic propositions $$\varphi \triangleq f(\mathbf x) > 0$$ we use Interval-Newton iteration.

$$ P \triangleq [\mathrm X] > 1 \qquad Q \triangleq [\mathrm X] < 2 $$
